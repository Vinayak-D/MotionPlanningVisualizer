Motion Planning: Dubai Map

Set start and goal locations, and visualize the A* Algorithm

Dubai Map

The A* Algorithm finds the shortest path from a start to goal location whilst avoiding obstacles.

Click on Set Start Location, double click anywhere in the white region, do the same for Set Goal Location. Then press Go! to find a path.

To quickly get started I recommend first setting the start and goal very close (200-300pixels apart). The further apart they are, the longer the algorithm takes to find a path. You can then proceed with calculating longer routes.

Be sure to Reset Grid between each calculation, Let's Go!
